- Java集合框架是什么？说出一些集合框架的优点？
>每种编程语言中都有集合，最初的Java版本包含几种集合类：Vector、Stack、HashTable和Array。  
随着集合的广泛使用，Java1.2提出了囊括所有集合接口、实现和算法的集合框架。在保证线程安全的情况下使用泛型和并发集合类，Java已经经历了很久。
